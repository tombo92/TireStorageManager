Positionsliste bearbeiten

Eine Position pro Zeile. Die Reihenfolge hier bestimmt die Sortierung überall in der Anwendung.
Sie können beliebige Namen verwenden (z.B. Halle-A-R1-O, Lager-3-Links, …).

Zurücksetzen

Setzt alle Positionen auf die vordefinierten Container + Garagen-Positionen zurück.

Tipps
  • Jede Zeile = eine Position
  • Leere Zeilen werden ignoriert
  • Die Reihenfolge bestimmt die Vorschlagspriorität
  • Bestehende Radsätze behalten ihre Position auch wenn sie hier entfernt wird
  • Für Container/Garage-Schema nutzen: C{1-4}{R/L}{O/M/U}{LL/L/MM/M/RR/R} bzw. GR{1-8}{O/M/U}{L/M/R}
